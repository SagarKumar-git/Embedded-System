# 📖 About

This repository contains all **20 experiments** performed during the **Cyber Physical Systems Laboratory** course using the **Arduino Nano 33 BLE Sense**.

The experiments cover:

- Embedded Programming
- Sensors
- IMU
- Gesture Recognition
- RGB LED
- I2C Communication
- BLE Communication
- Microphone
- RTOS
- Timer Interrupts
- Web Communication

---

# 🛠 Hardware Used

- Arduino Nano 33 BLE Sense
- USB Cable
- Grove LCD Display
- APDS9960 Gesture Sensor (Built-in)
- LSM9DS1 IMU (Built-in)
- Digital Microphone MP34DT05
- RGB LED (Built-in)
- PC/Laptop

# 📋 Lab Summary

## Lab 1
Configured Arduino Nano 33 BLE Sense using Arduino IDE.

---

## Lab 2
Learned digital output by blinking the onboard LED.

---

## Lab 3
Printed text on the Serial Monitor using Serial Communication.

---

## Lab 4
Controlled the onboard RGB LED and displayed different colours.

---

## Lab 5
Read Accelerometer values and displayed waveforms on Serial Plotter.

---

## Lab 6
Read Gyroscope values and observed rotational movement.

---

## Lab 7
Detected Left, Right, Front and Back tilt directions.

---

## Lab 8
Controlled RGB LED according to board tilt.

---

## Lab 9
Read Analog Input from pin A6 and converted it into voltage.

---

## Lab 10
Interfaced Grove I2C LCD and displayed messages.

---

## Lab 11
Detected UP, DOWN, LEFT and RIGHT hand gestures.

---

## Lab 12
Displayed gesture data on a web page using Python and Serial communication.

---

## Lab 13
Controlled RGB LED using proximity sensor values.

---

## Lab 14
Captured sound level using the onboard digital microphone.

---

## Lab 15
Controlled LED based on sound threshold.

---

## Lab 16
Blink LED using Bare Metal Timer Interrupt.

---

## Lab 17
Counted proximity events using timer.

---

## Lab 18
Performed basic RTOS task scheduling demonstration.

---

## Lab 19
Established Bluetooth Low Energy communication between Arduino and Mobile.

---

Controlled RGB LED from a mobile phone using BLE.
